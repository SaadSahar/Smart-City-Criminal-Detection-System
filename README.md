# Smart City Criminal Detection System - نظام مراقبة ذكي للمدينة

نظام مراقبة ذكي للمدينة يعتمد على الذكاء الاصطناعي لتحليل بث الكاميرات المنتشرة في الشوارع والأماكن العامة بشكل لحظي.

## المميزات | Features

- **اكتشاف الوجوه**: التعرف على الوجوه في الوقت الفعلي من كاميرات IP أو كاميرا الويب
- **قاعدة بيانات المطلوبين**: مقارنة الوجوه مع قاعدة بيانات صور المطلوبين
- **تنبيهات فورية**: إرسال تنبيهات عند اكتشاف شخص مطلوب مع رقم الكاميرا ووقت الالتقاط
- **حفظ الوجوه المجهولة**: حفظ صور الوجوه غير المعروفة تلقائياً
- **سجل قاعدة البيانات**: تسجيل كل عمليات التعرف في قاعدة بيانات SQLite
- **تصدير Excel**: تصدير سجلات التعرف إلى ملف Excel

## المتطلبات | Requirements

- Python 3.8+
- OpenCV
- face_recognition
- dlib
- pygame

## التثبيت | Installation

```bash
# إنشاء بيئة افتراضية
python -m venv .venv

# تفعيل البيئة
# Windows:
.venv\Scripts\activate
# Linux/macOS:
source .venv/bin/activate

# تثبيت المتطلبات
pip install -r requirements.txt
```

## الاستخدام | Usage

### 1. إعداد صور المطلوبين

ضع صور المطلوبين في مجلدات منفصلة:
```
data/wanted_faces/
├── أحمد_محمد/
│   ├── photo1.jpg
│   └── photo2.jpg
└── سعيد_علي/
    └── photo1.jpg
```

### 2. تشغيل النظام

```bash
python main.py
```

### 3. التحكم

- **ESC**: تصدير السجل إلى Excel والخروج
- **W**: مسح السجل

## الإعدادات | Configuration

يمكن تعديل الإعدادات في ملف `config.py`:

- `SOURCE_IS_IP`: True لكاميرا IP، False لكاميرا ويب
- `IP_URL`: رابط كاميرا IP
- `TOLERANCE`: عتبة التعرف (0.45 افتراضياً)
- `COOLDOWN_SECONDS`: مهلة التهدئة بين التنبيهات

## آلية العمل | How It Works

1. **تحميل الترميزات**: تحميل ترميزات وجوه المطلوبين من قاعدة البيانات
2. **التعرف اللحظي**: تحليل كل إطار من الكاميرا واكتشاف الوجوه
3. **المقارنة**: مقارنة الوجوه المكتشفة مع قاعدة بيانات المطلوبين
4. **التنبيه**: عند التعرف على مطلوب:
   - تشغيل صوت تنبيه
   - تسجيل في قاعدة البيانات
   - حفظ معلومات الكاميرا والوقت
